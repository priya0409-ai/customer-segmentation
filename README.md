# 🤖 Customer Segmentation using K-Means

## 🚀 Project Overview
This project performs customer segmentation using Machine Learning (K-Means Clustering). Customers are grouped based on their Annual Income and Spending Score.

## 📌 Features
- K-Means Clustering
- Customer grouping
- Data visualization using scatter plot
- Behavioral analysis

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📂 Dataset
The dataset contains:
- Age
- Annual Income
- Spending Score

## ▶️ How to Run
1. Install libraries:
   pip install pandas matplotlib scikit-learn

2. Run:
   python segmentation.py

![Customer Segmentation](output.png)

## 📈 Insights
- High income & high spending → Premium customers
- Low income & low spending → Budget customers
- Medium groups → Moderate customers
